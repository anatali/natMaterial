Start-up

  1. Connect to https://start.spring.io/
  2. Select Gradle Project, Java, Group=it.unibo, Artifact=HealthAdapterFacade (Options:Packaging=Jar, Java=8), Dependencies:
    • Spring Boot DevTools
    • Spring WebFlux //Reactive
    • Spring Web //Conventional: Not used at the moment
    • Thymeleaf //Not used for m2m interaction
    • Spring Security //Not used at the moment
    • Spring Redis //just to remember ...
  3. Click on Generate, unzip the generated file into your working dir and update build.gradle and application.properties (port=8081):
    • Set some logo for human-to-machine (h2m) interaction and for machine-to-machine (m2m) interaction
    • Enable/Disable security
    • Set endpoint management
    • Enables spring.jmx
  4. Use patorjk.com/software/taag to change the logo in file banner.txt

  5. Define in templates a GUI page: welcome.html for h2m interaction

  6. Define
    • a 'conventional' controller: HumanInterfaceController.java to be used with Spring Web and Thymeleaf
    • a RestController (see Build Reactive REST APIs With Spring WebFlux ): MachineInterfaceController.java to be used with Spring WebFlux for m2m interaction.
  7. Run HealthAdapterFacadeApplication.java, localhost:8081 and JConsole
